# cn-jinbaobo-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖特定后端服务，可直接通过浏览器访问。

## 项目简介

这是一个纯前端资源仓库，主要存储一些独立、自包含的 HTML 页面。每个页面通常包含完整的样式、脚本与结构，适合直接打开或嵌入其他场景使用。

仓库本身不绑定任何特定域名或网站，仅作为一个集中存放与版本管理的空间。

## 目录结构

```
cn-jinbaobo-resource/
├── pages/          # 存放独立 HTML 页面
├── assets/         # 页面共用资源（如样式、图片、脚本）
└── README.md       # 本说明文件
```

- `pages/`：每个子目录或文件对应一个独立页面，页面名称通常反映其功能或用途。
- `assets/`：存放多个页面可能共同引用的资源文件，避免重复。

目录结构可能随归档内容增加而调整，具体以仓库实际文件为准。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面之间相互独立，无跨页面依赖（除共用资源外）。
- 部分页面可能需要现代浏览器支持（如 ES6+、CSS3 特性）。
- 归档内容不涉及任何特定网站、服务或商业推广。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用浏览器打开 `pages/` 下的 HTML 文件即可查看对应页面。
3. 也可将整个仓库部署到任意静态服务器上，通过 URL 访问各页面。
